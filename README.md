# Study Planner Web App

A fully functional student study planner built using HTML, CSS, and Vanilla JavaScript.

## Features
- Add / Edit / Delete tasks
- Mark tasks as Done / Undo
- Overdue detection
- Filter (All / Pending / Completed)
- Search by task or subject
- Sort by Due Date / Newest / Oldest
- Persistent storage using LocalStorage

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose
Built as a frontend portfolio project to strengthen practical web development skills.
